# Tiny Dungeons 2e
### Test
- Způsob určení úspěchu/ neúspěchu u komplexnějších akcí
- *2d6*
	- Pokud padne alespoň 1x *5*, nebo *4* => úspěch
	- Výhoda = *3d6* (max 1x, takže žádné 4d6)
	- Nevýhoda = *1d6*
		- zdroje = situační, schopnosti, magické předměty
	- Nevýhoda přepíše výhodu
### Překážka
- Situace, kterou se skupina snaží vyřešit
- např. smlouvání s obchodníkem, odemykání zámku, hledání předmětu,...
- Dobrý rp, nebo příprava může automaticky uspěch
- jinak je potřeba řešit Testem
